# 克隆声音使用指南

## 安装完成后怎么用

你只需要给 Hermes 发送两样东西：

1. 一段本人授权的参考声音；
2. 一段要朗读的文字。

推荐说法：

```text
用我的声音朗读下面这段文字。
参考声音我随后发给你。
文字是：大家好，欢迎来到今天的课程。
```

或者：

```text
我发一段录音和一段文案，请你克隆这个声音来朗读文案。
```

## 参考声音要求

- 建议 5-15 秒；
- 环境安静；
- 只有一个人说话；
- 不要有背景音乐；
- 使用本人或已获得授权的声音。

## 下次再次使用

如果已经安装过，不需要重新安装。直接说：

```text
用我的声音朗读这段文字。
```

然后发送参考声音和文字即可。

如果你想长期固定使用同一段声音，可以让 Hermes 保存为默认参考声音；之后只发文字即可。
